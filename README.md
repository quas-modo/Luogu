## 最短路
### Dijkstra总结
1. 找到点（判断是否已访问）
2. 更新访问状态，即将点放入S点集中
3. 更新相连接的点的最短

## 最小生成树

边权最小的边一定在mst中
### prim
1. 找到点（判断是否已访问）
